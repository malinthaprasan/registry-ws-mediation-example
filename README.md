SOAP Based Client for updating a mediation policy using WSRegistryService SOAP API.

Tested for API Manager - 1.10.0

To run the sample,

Clone the repository
Copy "registry-ws-mediation-example" to APIM_HOME/samples folder
Run "ant run".
